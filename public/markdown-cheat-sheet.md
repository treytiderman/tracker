# Markdown Cheat Sheet (Heading 1)

To create paragraphs, use a blank line to separate one or more lines of text.

- Basic Syntax
    1. Italic *text*
    2. Bold **text**
    3. Code `text`
- Extended Syntax
    1. ~~Strikethrough text~~
    2. ==Highlight text==
    3. Subscript H~2~O
    4. Superscript X^2^
